# Add shadcn-ui Component

Add a new shadcn-ui component to the project.

## Usage
Provide the component name you want to add.

## Available Components
Common shadcn-ui components not yet installed:
- calendar
- carousel
- chart
- combobox
- command
- data-table
- date-picker
- drawer
- sonner
- toast

## Command to Add
```bash
npx shadcn-ui@latest add <component-name>
```

The component will be added to `src/components/ui/`.

Ask the user which component they want to add, then run the command.
